Class WorkflowHandleService

The WorkflowHandleService provides methods to interact with a running workflow. This includes exporting the workflow, sending signals, and querying the state of the workflow. It is instanced/accessed via the MemFlow.Client class.

Example

import { Client } from '@hotmeshio/hotmesh';
import { Client as Postgres } from 'pg';

const client = new Client({ connection: {
  class: Postgres,
  options: { connectionString: 'postgres://user:pass@localhost:5432/db' }
}});

const handle = await client.workflow.start({
 args: ['HotMesh'],
 taskQueue: 'hello-world',
});

//perform actions like send a signal
await handle.signal('my-signal', { data: 'Hello' });
