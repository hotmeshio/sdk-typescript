Interface UpdateEscalationParams

Fields that can be patched on an existing escalation. All fields are optional — only provided fields are written. Signal routing fields (signalKey, topic, workflowId, taskQueue, workflowType) support the legacy two-step pattern where routing context is enriched after creation.
